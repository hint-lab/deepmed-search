## 🚀 如何使用

### 访问文档中心

从项目 README 进入：
- **中文**: [README.zh-CN.md](./README.zh-CN.md) → 点击"文档中心"
- **English**: [README.md](./README.md) → Click "Documentation Center"

或直接访问：
- **中文**: [docs/README.md](./docs/README.md)
- **English**: [docs/README.en.md](./docs/README.en.md)

### 快速开始

1. **部署项目**
   ```bash
   # 查看部署检查清单
   cat docs/deployment/DEPLOYMENT_CHECKLIST.md
   
   # 使用 SSL 快速启动
   ./docs/deployment/setup-ssl.sh
   ```

2. **配置功能**
   ```bash
   # 浏览设置指南
   ls docs/setup/
   ```

3. **开发新功能**
   ```bash
   # 查看开发文档
   ls docs/development/
   ```

### 检查文档链接

运行自动化检查工具：

```bash
./docs/check-links.sh
```
